##  Dynamic Scope And Lexical Scope
```
const a = function()
{
  console.log('a',this);
  const b = function()
  {
    console.log('b',this);
    const c ={
      hi: function()
      {
        console.log('c',this)
      }
    }
    c.hi()
  }
  b()
}
a()


//a window Object
//b window Object
//c {hi :f }

```

- In modern JavaScript (ES6+), prefer arrow functions when you want to preserve the context (this) of the outer function â€” they make code cleaner and easier to reason about.
- Arrow function has lexical scope which come in Es6 in above code we can create arrow function insted of calling another function. 
- just doing Arrow function we call reduce confusion
